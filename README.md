# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`npm install github:barrysteyn/node-scrypt#fb60a8d3c158fe115a624b5ffa7480f3a24b03fb`

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
